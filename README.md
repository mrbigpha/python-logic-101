## 🛠 วิธีการใช้งานและส่งงาน
1. **Fork** โปรเจกต์นี้ไปที่ Account ของตัวเอง
2. **Clone** ลงเครื่องคอมพิวเตอร์:
   ```bash
   git clone [URL-ที่-Fork-มา]
   ```
3. **สร้าง Branch ใหม่** เพื่อแยกงาน:
   ```bash
   git checkout -b [ชื่อ-Branch-ของคุณ]
   ```
4. **แก้ไขไฟล์** และทดสอบ:
   ```bash
   python [ชื่อไฟล์].py
   ```
5. **Commit** การเปลี่ยนแปลงของคุณ:
   ```bash
   git add .
   git commit -m "[ข้อความอธิบายการเปลี่ยนแปลง]"
   ```
6. **Push** การเปลี่ยนแปลงไปยัง GitHub:
   ```bash
   git push -u origin [ชื่อ-Branch-ของคุณ]

7. **Notify** แจ้งอาจารย์ผู้สอนว่าได้ส่งงานแล้ว โดยก็อปปี้ ลิงค์ Repo ของตัวเองจาก GitHub ไปให้อาจารย์ผู้สอน
