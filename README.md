## 🛠 วิธีการใช้งานและส่งงาน
1. **Fork** โปรเจกต์นี้ไปที่ Account ของตัวเอง
2. **Clone** ลงเครื่องคอมพิวเตอร์:
   ```bash
   git clone [URL-ที่-Fork-มา]
   ```
3. **สร้าง Branch ใหม่** เพื่อแยกงาน:
   ```bash
   git checkout -b [ชื่อ-Branch-ของคุณ]
   ```
4. **แก้ไขไฟล์** และทดสอบ:
   ```bash
   python [ชื่อไฟล์].py
   ```
5. **Commit** การเปลี่ยนแปลงของคุณ:
   ```bash
   git add .
   git commit -m "[ข้อความอธิบายการเปลี่ยนแปลง]"
   ```
6. **Push** การเปลี่ยนแปลงไปยัง GitHub:
   ```bash
   git push -u origin [ชื่อ-Branch-ของคุณ]

7. **Notify** แจ้งอาจารย์ผู้สอนว่าได้ส่งงานแล้ว โดยก็อปปี้ ลิงค์ Repo ของตัวเองจาก GitHub ไปให้อาจารย์ผู้สอน

## 💡 คำแนะนำในการทำโจทย์

ในไฟล์โจทย์ทุกข้อจะมีคำสั่ง `raise NotImplementedError` อยู่ ซึ่งเป็น "สัญญาณ" บอกระบบตรวจว่าคุณยังไม่ได้เริ่มทำข้อนั้น

### 🛠 ขั้นตอนการเริ่มทำ:
1. **เปิดไฟล์โจทย์** (เช่น `ex01_is_even.py`)
2. **ลบบรรทัด** `raise NotImplementedError` ออก
3. **เขียนโค้ด** ของคุณลงไปในช่องว่างระหว่าง `# --- เริ่มเขียนโค้ดด้านล่างนี้ ---` และ `# --- สิ้นสุดการเขียนโค้ด ---`
4. **ตรวจสอบ** โดยการรันไฟล์ตรวจสอบหลัก:
   ```bash
   python3 check_solution.py
   ```

### 📊 สถานะในระบบตรวจ:
*   ⏳ **ยังไม่ได้เริ่มทำ**: คือข้อที่มี `raise NotImplementedError` อยู่ (จะไม่ถูกนับเป็นข้อที่ผิด)
*   ✅ **ผ่าน**: โค้ดทำงานถูกต้องตาม Test Cases ทั้งหมด 5 ชุด
*   ❌ **ไม่ผ่าน**: โค้ดทำงานผิดพลาด หรือมี Logic ที่ไม่ตรงตามเงื่อนไข (ระบบจะแจ้งว่าผิดที่ชุดข้อมูลไหน)
